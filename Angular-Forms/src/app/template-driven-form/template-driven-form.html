Validation Attribute Purpose
required Makes the field mandatory.
minlength="number" Sets minimum length of input.
maxlength="number" Sets maximum length of input.
min="number|date" Minimum numeric or date value allowed.
max="number|date" Maximum numeric or date value allowed.
step="number" Specifies intervals between legal numbers.
pattern="regex" Input must match this regular expression pattern.
type="email" Validates email format.
type="url" Validates URL format.
type="number" Accepts only numeric input.
type="tel" Validates telephone number loosely.
readonly Field is read-only but submitted.
disabled Field is disabled, not submitted.
multiple Allows multiple selections or inputs.
